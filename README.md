# Python Flet Wi-Fi Demo

这是一个基于 Python 和 Flet 框架构建的跨平台 Wi-Fi 管理器演示项目。

## 简介

本项目旨在演示如何使用 Flet 构建一个具有现代化 UI 的跨平台应用程序，用于扫描和管理 Wi-Fi 网络。

## 快速开始

### 1. 环境准备

确保您的系统中已安装 Python 3.x。

### 2. 安装依赖

安装 Flet 及相关依赖：

```bash
pip install flet
```

### 3. 运行项目

推荐使用虚拟环境来运行本项目。执行以下命令进入项目目录、激活虚拟环境并启动应用：

```bash
cd "/Users/hooksvue/Desktop/trae solo ai -demo-flet" 
source venv/bin/activate 
python main.py
```

## 文件结构

- `main.py`: 应用程序主入口
- `wifi_scanner.py`: Wi-Fi 扫描相关核心逻辑
